# DigitalBoost AI — Produits Digitaux Premium

Site de vente de produits digitaux (eBook, Prompts IA, Templates) construit avec HTML, CSS et JavaScript vanille.

## Structure
- `index.html` — Landing page principale
- `products/` — Pages produits individuelles
- `ebook/` — Contenu de l'eBook (produit livrable)
- `prompts/` — Pack de prompts IA (produit livrable)

## Déploiement
Hébergé sur Netlify. Déploiement automatique via `netlify deploy --prod`.
